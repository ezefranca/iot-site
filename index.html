O que é?

O equipamento

Baseado na plataforma de prototipagem eletrônica livre Arduíno, foi desenvolvido um shield com sensores de luminosidade, contato, umidade, ruído e luz.

A linguagem

Uma biblioteca em JavaScript (node.js) foi criada para simplificar o acesso aos sensores pelos desenvolvedores e enviar os dados diretamente a uma plataforma baseada na nuvem.

A nuvem

Os dados são enviados a uma plataforma de alta performance escalável, que foi desenvolvida para ser flexível e modular para facilitar a construção de aplicações M2M baseadas em dados em tempo real.

Instalalção

Windows OBS: O kit é compatível apenas com o Windows 7 SP1 ou superior!

Softwares necessários:

  1. Instalar software Arduino

    http://arduino.cc/en/Main/Software#toc2

  2. Visual Studio 2013 Express para Windows Desktop (ou superior):

    http://www.visualstudio.com/downloads/download-visual-studio-vs#d-2013-express

  3. Node.JS 0.10 ou superior:

    http://nodejs.org/download/

  4. Python 2.7.6 ou superior:

    http://www.python.org/download/releases/2.7.6/

Depois de instalado

Para instalar o Kit IoT, conecte o cabo usb no computador e na placa arduino e digite no terminal:

sudo npm install kit-iot -g

Para inicia-lo novamente, digite no terminal:

sudo kit-iot

Se for WINDOWS não utilize sudo

API rest

Comando

GET services/{token}/

Descrição

Informações detalhadas do serviço

Endereço URL

http://dca.telefonicabeta.com/m2m/v2/services/{token}/

Requer Autenticação?

SIM

Parâmetros

Nenhum

GET services/{token}/
assets/{token}/

Retorna informações de um determinado asset de um determinado serviço

Endereço URL

http://dca.telefonicabeta.com/m2m/v2/services/{token}/assets/{token}/

Requer Autenticação?

SIM

Parâmetros

Nenhum

GET services/{token}/
assets/{token}/data/

Retorna o histórico de dados recebidos de um determinado asset de um determinado serviço

Endereço URL

http://dca.telefonicabeta.com/m2m/v2/services/{token}/assets/{token}/data/

Requer Autenticação?

SIM

Parâmetros

limit *opcional

Total de registros para retornar por página. Caso não seja especificado, a API retornará todos os dados recebidos

sortBy *opcional

Ordena os dados (ascendente ou descendente) utilizando o campo informado. Os campos disponíveis são: assetName, assetType, ou registrationTime. Exemplos:
sortBy=registrationTime
sortBy!=registrationTime
