O que é?

O equipamento

Baseado na plataforma de prototipagem eletrônica livre Arduíno, foi desenvolvido um shield com sensores de luminosidade, contato, umidade, ruído e luz.

A linguagem

Uma biblioteca em JavaScript (node.js) foi criada para simplificar o acesso aos sensores pelos desenvolvedores e enviar os dados diretamente a uma plataforma baseada na nuvem.

A nuvem

Os dados são enviados a uma plataforma de alta performance escalável, que foi desenvolvida para ser flexível e modular para facilitar a construção de aplicações M2M baseadas em dados em tempo real.

Comando

GET user/login

Descrição

Autenticação do usuário

Endereço URL

http://195.235.93.67:8080/secure/m2m/v2/user/login

Cabeçalhos

Authorization *obrigatório

M2MUser {user}:{password}

Parâmetros

organization *obrigatório

Sempre fornecer organization=1

permissions *obrigatório

Sempre fornecer permissions=1

Parâmetros

GET services/{serviceName}/
assets/{assetId}/

Retorna informações de um determinado asset de um determinado serviço

Endereço URL

http://195.235.93.67:8080/secure/m2m/v2/services/{serviceName}/assets/{assetId}/

Requer Autenticação?

SIM

Parâmetros

Nenhum

GET services/{serviceName}/
assets/{assetId}/data/

Retorna o histórico de dados recebidos de um determinado asset de um determinado serviço

Endereço URL

http://195.235.93.67:8080/secure/m2m/v2/services/{serviceName}/assets/{assetId}/data/

Requer Autenticação?

SIM

Parâmetros

limit *opcional

Total de registros para retornar por página. Caso não seja especificado, a API retornará todos os dados recebidos

sortBy *opcional

Ordena os dados (ascendente ou descendente) utilizando o campo informado. Os campos disponíveis são: assetName, assetType, ou registrationTime. Exemplos:
sortBy=registrationTime
sortBy!=registrationTime
